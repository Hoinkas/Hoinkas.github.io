    #4################ INSTRUKCJE WARUNKOWE ################
    '''
    Pętle w Pythonie są używane do wykonywania tych samych instrukcji wielokrotnie, aż do spełnienia określonego warunku. 
    Jeśli warunki nie są spełnione, to kod nie jest wykonywany i przechodzi do następnego warunku lub do kodu, jeśli nie ma więcej warunków.
    '''
    
    # --------
    '''
    Pętla if, wykonuje się, gdy warunek jest prawdziwy.
    Jest to pierwszy rodzaj pętli warunkowej, od którego zawsze zaczynamy sprawdzanie warunków.
    Może być tylko jeden if na daną sytuację.
    '''

    if a > b:
      print("a jest większe od b");

    # --------
    '''
    Drugim rodzajem jest pętla elif, która wykonuje się, gdy poprzedni warunek nie jest prawdziwy.
    Podajemy w niej warunek do spełnienia. 
    ifelse może być wiele.
    '''

    if a > b:
      print("a jest większe od b");

    elif a < b:
      print("a jest mniejsze od b");

    elif a == b:
      print("a jest równe b");

    # --------
    '''
    Trzecim rodzajem jest pętla else, która wykonuje się, gdy żaden z poprzednich warunków nie jest nie jest prawdziwy. 
    Nie podajemy w niej warunku.
    '''

    if a > b:
      print("a jest większe od b");

    elif a < b:
      print("a jest mniejsze od b");

    elif a == b:
      print("a jest równe b");

    else:
      print("a jest równe b");

    '''
    Pętla while wykonuje się tak długo jak długo spełniony jest warunek. W tym przypadku i zaczyna od 0 i kończy na 5.
    W tym przypadku pod koniec każdej pętli i jest zwiększane o 1.
    Pętla sprawdza przed każdym wykonaniem czy warunek jest spełniony. 
    
    W pętli piątej, kiedy x = 5 i pod koniec zmieni się na x = 6 przy pomocy x += 1 to while sprawdza warunek i widzi że nie jest spełniony i kończy pętlę.
    '''
    i = 0;
    while i < 6:
